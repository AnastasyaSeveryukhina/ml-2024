## Изменения в проекте с 07.10 по 13.10

1) Изменен формат хранения координат в обучающей выборке модели. Теперь там хранятся не долгота и широта в географических координатах, а локальные декартовые прямоугольные координаты с центром в точке углу береговой линии. Это сделано для того, чтобы входные и выходные данные из модели были более чувствительным к небольшим смещениям якорей на карте. Также изменена запись обучающих данных для модели с глобальных координат в локальные.

2) Создана небольшая обучающая выборка. Для этого использовался модуль для записи обучающих данных, созданный на прошлой неделе.

См. ML_data.dat

3) Сформулирована и построена простейшая модель линейной регрессии для предсказания координат якорей. Использовлся пакет sklearn и метод LinearRegression()

Входные данные модели: 4 числа - координаты двух угловых точек желаемой рабочей зоны

Выходные данные модели: 16 чисел - координаты 8 якорей в локальной системе координат.

Результаты получились выше среднего относительно ожиданий :) на примерах, близких к тестовому, модель дает хорошие резульаты. Рабочая область, построенная по координатам якорей, часто содержит внутри себя желаемую область проведения работ. На результатах, отличных от тестовых, результаты сильно хуже - часто якоря оказываются в хаотичном порядке, нет четкой структуры их расположения
